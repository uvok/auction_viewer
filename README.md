# Auction Viewer

WIP.
C++ GUI application to perform a "stage auction".
App will pull AuctionItems from a yet-to-be-defined API.
MainWindow will present the list of auction items on one side, an editor for the currently selected item on the other side.
AuctionItems have Title, StartPrice,  CurrentPrice, AuctionWinner.
MainWindow will be for person presenting the auction. User of the application will enter currently highest bid manually.

There will also be a PresenterWindow for the audience screen.
This will show the title, the current bid, as well as an optional image of the item.
